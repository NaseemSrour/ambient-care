TRMNL Arabic Memory Day-Clock

A customized cognitive support dashboard for elderly family members

🧠 Cognitive Care Visual Techniques

🌙

1. Color-Inverted Night Mode

The Technique: Automatically switch the interface to a pitch-black background with clean white Arabic text after 8:00 PM, changing the central text to "وقت النوم" (Bedtime).

The Benefit: Manages "Sundowning" anxiety and evening confusion. It establishes a strong visual boundary that communicates the day has safely ended without glowing blue light.

📋

2. Visual Routine Checklists

The Technique: Feed daily routine blocks into the layout framework via your Python payload using simple static checkmark graphics.

The Benefit: Strips away rigid timeframes that cause time-blindness panics. Replaces them with clear contextual signals like "[✓] تناول دواء الصباح" (Morning medicine taken).

⬛

3. Hyper-Simplified Spatial Orientation

The Technique: Clean the HTML canvas entirely of status elements, batteries, or smaller numbers. Use only three stark, massive text components separated by solid lines.

The Benefit: Prevents sensory overstimulation. It ensures a disoriented user can instantly determine the day, broad time frame, and date without processing non-essential data filters.

🖼️

4. Over-the-Air Reassurance Messages

The Technique: Use a Python text queue to allow family members to temporarily override the clock interface with custom messages like "نحن نحبك يا جدي" (We love you, Grandpa).

The Benefit: Combats isolation-induced anxiety dynamically. It refreshes their room environment with familiar names, keeping them grounded and connected without any device interaction required on their part.

🚀 Development & Deployment Framework

  1. Build Your Server Architecture (Python/FastAPI)

    Set up a Python backend endpoint that formats the current time parameters into structured Arabic text variables. Program your server logic to check a storage JSON payload first; if a family member has submitted a custom message through your Flutter app, prioritize displaying the message text over the standard time fields.

  2. Configure Your Private Plugin

    Log into your TRMNL online developer profile. Navigate to Plugins > Private Plugins > Create New and set the data architecture source to Polling. Paste your hosted Python backend URL into the configuration fields and set the check-in frequency timer.

  3. Inject the Visual Markup Template

    Open the Markup code environment tab inside your new plugin console. Embed a robust, clean web font (such as Cairo or Amiri) using an explicit external stylesheet link. Use standard right-to-left layout controls (direction: rtl; text-align: right;) alongside responsive Liquid template tags to map your Python script variables cleanly across the monochrome viewport layout.

  4. Deploy Over-The-Air (Zero Physical Contact)

    Add your completed custom plugin directly onto your TRMNL profile dashboard Playlist. The hardware terminal will pull the changes over your home Wi-Fi during its next scheduled background check-in. Any subsequent updates to your Python server logic, layout files, or incoming family texts deploy completely remotely without ever touching the physical machine.

⚡ Hardware Design Reminder: The TRMNL screen functions like authentic print on paper and does not glow in the dark. This guarantees zero circadian rhythm disruptions for your relative. To enable near-instant 5-minute update feeds for family messages without draining the battery, plan to keep the hardware connected to a permanent USB-C power source.
